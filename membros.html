  • Projeto:
      Fizemos uma página sobre ranking de séries, iniciando com um login que direciona para o menu inicial, após o direcionamento para o menu inicial, na parte superior da página terá acesso a todas as categorias: gêneros, contato, membros e voltar ao login.

      Na página do menu incial terá uma mensagem sobre a página

      Nas páginas de ação, terror e animação terá séries sobre cada gênero e o usuário poderá dar a nota que achar melhor para cada uma e em cima de casa imagem terá uma mensagem "link" direcionando de onde baixamos cada imagem.

      Na página de contato terá um formulário que o usuário preencherá para entrar em contato com os criadores da página

      Após preencher o formulário de contato irá direcionar para uma página que irá deixar detalhado que foi preenchido.

      Na página dos membros terá a descrição de quem participou do projeto e o que foi feito nele.


  • O que foi utilizado referente ao PHP e Formulários:
      Na página de login, utilizamos o session-start para iniciar uma sessão, criamos duas variáveis para usuário e senha e utilizamos o método POST para proteger as informações, if para determinar o nome de usuário e a senha para o acesso ao site e Else para determinar uma mensagem de erro caso o usuário não preencha corretamente.

      Já no Index utilizamos a função If Isset para verificar se já foi preenchido o formulário para não apresentar mensagem de erro, caso seja preenchido corretamente apresenta uma mensagem de bem vindo e um link de direcionamento ao menu inicial e o Else para apresentar uma mensagem de erro caso usuário ou senha estejam incorretos.

      Para preencher usuário e senha criamos um formulário somente com usuário e senha e um submit para enviar ao login.

      Nas categorias: Ação, Terror e Animação, utilizamos If Isset para varificar se já foi preenchido o formulário para não apresentar mensagem de erro, criamos uma variável para armazenar a nota determinada pelo usuário, para isso utilizamos o switch case e determinamos 5 casos e para cada caso uma mensagem referente a nota determinada pelo usuário, utilizamos o break para apresentar somente a mensagem referente a nota que o usuário deu para a série e para ele conseguir avaliar utilizamos um form e para cada form utilizamos o option e em vez de utilizar o option com números, determinamos o valor por estrelas e limitamos a 5 estrelas no máximo, após escolha da nota um submit para enviar a nota.

      No formulário de contato utilizamos form para nome, e-mail, telefone e mensagem e um submit para enviar os dados, após preencher e enviar todos os dados criamos uma página para a mensagem que foi enviada pelo formulário de contato, nesta página criamos uma variável para cada form (nome, e-mail, telefone e mensagem) utilizando o metodo POST para deixar as informações mais seguras, e um echo para mostrar a mensagem que foi enviada.
